Function file to output a duty cycle and direction to the motor driver.

This function outputs the user-specified PWM duty cycle and direction to the motor using analogWrite() and digitalWrite(). The direction pin is essentially a boolean. Writing a 0 will drive the motor in the forward direction, and writing a 1 will drive the motor in reverse. However, this function takes the direction as an int23_t instead of a boolean because function, parseIntWithEcho() returns user-input numbers as int32_t.
